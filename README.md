# Astro SSR Playground

This repository is a small playground for exploring Astro with Tailwind CSS. Nothing here is final - it's just a space to try things out and experiment.

## Headless CMS

[![minima.ltd](https://img.shields.io/badge/minima.ltd-Headless%20CMS-blue)](https://minima.ltd)

## Live Demo

[Live Demo](https://astro-ssr-playground.pages.dev/)

## Stack

- Astro
- Tailwind CSS
- Minima CMS

## Installation

```
pnpm install
```

## Scripts

```bash
pnpm dev      # Start dev server
pnpm build    # Build for production
```
